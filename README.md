## 🚀 Considerações iniciais.

Software desenvolvido para exibir um gui de personagens da franquia STARWARS **React.js** para o frontend.

## Ambiente
- S.O: Debian GNU/Linux 10 (Buster)
- Node: v12.16.1
- npm: 6.13.4
- npx: 6.13.4
- yarn: 1.22.4
